# Cheat sheet de métodos de array

## SOME()

```js
[🏀, 🏓, 🍎, 🍕, 🥝, 🐳, 🍎].some(🍕) -> true
```

## EVERY()

```js
[🏀, 🏓, 🍎, 🍕, 🥝, 🐳, 🍎].every(🍎) -> false;
```

## FIND()

```js
[🏀, 🏓, 🍎, 🍕, 🥝, 🐳, 🍎].find(🥝) -> 🥝;
```

## FILTER()

```js
[🏀, 🍕, 🍎, 🍕, 🍕, 🐳, 🍎].filter(🍕) -> [🍕, 🍕, 🍕];
```

## FOREACH()

```js
[2, 4, 6, 8, 10].forEach(* 2) -> [4, 8, 12, 16, 20]
```

## SORT()

```js
[7, 5, 8, 3, 2, 1, 4, 6].sort() -> [1, 2, 3, 4, 5, 6, 7, 8]
```

## MAP()

```js
[🐮, 🥔, 🌽, 🪵, 🍎].map(🏭) -> [🍔, 🍟, 🍿, 🪑, 🥧]
```

## REDUCE()

```js
[🍔, 🍟, 🍿, 🥧, 🍩].reduce(🍽) -> 💩
````
